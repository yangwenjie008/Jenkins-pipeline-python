# Jenkins Pipeline as Code Project

This project demonstrates how to create a Jenkins Pipeline as Code solution with:
1. Groovy-based Jenkins job framework definition
2. Python-based implementation for specific functionality

## Project Structure

- `Jenkinsfile` - Main pipeline definition in Groovy
- `jobs/` - Directory containing job templates
- `scripts/` - Python scripts implementing specific functionality
- `pipelines/` - Groovy pipeline definitions

## Getting Started

1. Clone this repository
2. Configure Jenkins to use this repository
3. Customize the Jenkinsfile and Python scripts as needed
